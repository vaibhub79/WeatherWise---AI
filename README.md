# WeatherWise---AI
Modern AI-powered weather application with forecasts, AQI, voice assistant, GPS tracking, and responsive UI.

## Features

* Real-time Weather
* GPS Location Weather
* 24 Hour Forecast
* 7 Day Forecast
* Favorite Cities
* Search History
* AI Weather Summary
* Air Quality Index (AQI)
* Travel Score
* Dark Mode
* Voice Search

## Tech Stack

* HTML5
* CSS3
* JavaScript
* OpenWeather API
* Open-Meteo API
* Local Storage
* Geolocation API
